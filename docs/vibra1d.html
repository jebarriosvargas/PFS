4.1 Relaci√≥n de dispersi√≥n

\[ \definecolor{azul}{RGB}{0, 0, 139} \newcommand{\kk}{\textcolor{red}{k}} \]

\[ \omega^2 = 2\frac{\kappa }{m}\bigg[ 1-\cos(\kk a ) \bigg] \]

Entonces, \[ \omega(k) = \sqrt{\frac{4\kappa}{m}} \bigg| \sin\bigg( \frac{\kk a}{2} \bigg) \bigg| \]
